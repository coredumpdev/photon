Photon — GeoJSON vector map, no tiles / no server / no key

The whole map is one GeoJSON file rendered by Photon: lon/lat projected to Web Mercator, polygons triangulated (earcut), borders as thick lines, all on WebGL2. Drop in any admin-boundary GeoJSON (country → province → district). Here: Natural Earth countries, fetched at load. Drag to pan, wheel to zoom, click a country to read its name.
