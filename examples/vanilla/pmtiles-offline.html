Photon — offline vector map from a local .pmtiles

Pick a .pmtiles file from disk and it renders entirely offline — no server, no network, no API key. Photon reads byte ranges straight from the file via Blob.slice(), decodes MVT, triangulates and draws on WebGL2. Get a planet or region file from protomaps.com (or build your own with tippecanoe). Whole planet w/ buildings ≈ 100 GB; a country ≈ a few GB. Drag to pan, wheel to zoom, click a feature for its properties.
